DocForge Docs

Overview

DocForge is a hosted document engine. Template once, fill in data, and render print-grade PDFs — employee letters, GST invoices, and contract correspondence — with AI drafting and optional e-signing.

What is DocForge?

DocForge treats a document as structured data rendered through a template, not a Word file you edit by hand. Separate the two and everything improves: the template defines layout, branding, and logic once; the data fills it in; and the same template renders a thousand correct documents with no stray values from a previous draft.

It ships with ready-to-use packs — 11 HR letters, a GST-aware invoice engine, and contract-management letters — so most teams never author a template to get started.

How it works

1
Template
Install a built-in pack or author your own in the Studio with live preview. Set branding and a default watermark once.
2
Render
Fill the fields — or build an invoice with live totals — and render a pixel-perfect PDF. Every render is immutable and numbered.
3
Deliver & sign
Download it, or send a secure signing link. The signer reviews and signs; you get a sealed, certificate-backed PDF with a full audit trail.

Key capabilities

Template Studio

Author HTML/CSS + Jinja with a live split preview, rendered to PDF by WeasyPrint. Sandboxed — authors get templating, never arbitrary code. See Template Studio.

Invoice engine

Line items, GST (CGST/SGST or IGST), discounts, and totals with gap-free numbering. See Invoice Engine & GST.

HR & contract letters

11 employee letters and a set of contract covers, India-first and parameterised for global use. See Employee Letters.

AI drafting & e-signing

Draft letter bodies and clauses with Claude, and seal signed PDFs with a certificate page and SHA-256. See AI Drafting and E-Signing.
