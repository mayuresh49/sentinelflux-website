## What is SentinelFlux?

SentinelFlux generates test suites from your knowledge base, heals broken locators automatically, and analyzes every failure with AI — covering 8 testing modules from one dashboard.

- **AI Test Pipeline** — YAML knowledge base → test case docs → executable pytest scripts
- **Self-Healing Locators** — 3-tier fallback with AI re-discovery and human approval queue
- **11 AI Agents** — post-suite analysis: failure classification, flaky detection, regression guard, coverage gap, locator healing, strategic insights
- **8 Testing Modules** — REST/GraphQL, Web UI, VAPT (OWASP A01–A10), Performance, Accessibility, API Contract, Visual Regression, Bug Tracker
- **Remote Runner** — decoupled test execution via pull-based daemon; Playwright runs on your machine, results stream to the dashboard
