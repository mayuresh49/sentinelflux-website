On Screen

Voiceover

"VigilQA's accessibility module covers the four WCAG pillars: Perceivable — colour contrast, image alt text, media captions. Each page gets a detailed violation list with severity and remediation guidance."

VigilQA navigates to each page in a real Chromium browser, runs the axe-core accessibility engine plus its own checks, and collects every violation. Scene 4 — Results view1:35 – 2:25

On Screen

Accessibility results page. Show: overall score per page, violation count by severity (critical, serious, moderate, minor). Click a critical violation — shows: affected element (highlighted in screenshot), violation description, WCAG criterion reference, and suggested fix.

Critical violations are things that block users with disabilities completely — missing form labels, images without alt text, buttons with no accessible name. Each violation links to the affected element in a screenshot, shows the WCAG criterion it violates, and suggests a specific code fix. No manual interpretation required."
